# Карьерный советник (Causal Career Advisor)

Интерактивное веб-приложение для оценки причинно-следственных эффектов карьерных решений с помощью CausalForestDML.

## Возможности
- **36 карьерных действий**: от смены профессии до релокации за рубеж.
- **Персонализированные прогнозы**: baseline предсказывается отдельной моделью, затем добавляется CATE для каждого действия.
- **Горизонты прогноза**: 6 месяцев, 1 год, 2 года — модель обучается на всех периодах.
- **Сравнение сценариев**: таблицы, радарные диаграммы и графики динамики зарплаты.
- **SHAP-интерпретации**: быстрые текстовые пояснения и подробные waterfall-графики.
- **Качество модели**: метрики MSE, R², Pearson R и калибровочная кривая на тестовых данных.
- **Сохранение/загрузка модели**: кэширование обученной модели на диск.
