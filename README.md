# Карьерный советник (Causal Career Advisor)

Интерактивное веб-приложение для оценки причинно-следственных эффектов карьерных решений с помощью CausalForestDML.

## Возможности
- **30 карьерных действий**: от смены профессии до релокации за рубеж.
- **Персонализированные прогнозы**: baseline предсказывается отдельной моделью, затем добавляется CATE для каждого действия.
- **Горизонты прогноза**: 6 месяцев, 1 год, 2 года — модель обучается на 12 исходах (4 метрики × 3 горизонта): зарплата, удовлетворённость, вероятность повышения, work-life balance.
- **Сравнение сценариев**: таблицы, радарные диаграммы и графики динамики.
- **SHAP-интерпретации**: waterfall, summary и bar-графики важности признаков через `interpreter.py`.
- **Качество модели**: метрики MSE, R², Pearson R, placebo-тест и калибровочная кривая.
- **Два режима обучения**: быстрый (`fast_mode=True`, ~секунды) и точный (`high_precision=True`, несколько минут).
- **Сохранение/загрузка модели**: кэширование обученной модели на диск через `joblib`.

## Структура проекта

```
causal_decision_system/
├── app.py                  # Streamlit-приложение (главный файл)
├── causal_model.py         # CausalForestDML — обучение и предсказание CATE

├── prediction_service.py   # Сервисный слой: признаки, горизонты, SHAP
```

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repo-url>
   cd causal_decision_system
   ```

2. Создайте виртуальное окружение Python 3.10+:
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux / macOS
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите приложение:
   ```bash
   streamlit run app.py
   ```

## Быстрый старт (smoke-тест)

```bash
python test_model.py
```

Генерирует синтетические данные (n=2000), обучает модель в быстром режиме и выводит метрики качества.

## Режимы обучения

| Режим | Параметр | Время | Применение |
|-------|----------|-------|------------|
| Быстрый | `fast_mode=True` (по умолчанию) | секунды | разработка, демо |
| Точный | `high_precision=True` | несколько минут | продакшн |

## Примечания

- Данные синтетические — модель демонстрирует архитектуру каузального вывода, а не реальные карьерные закономерности.
- `inference=False` в быстром режиме отключает расчёт доверительных интервалов при обучении; в этом случае нижняя и верхняя границы интервала совпадают с точечной оценкой.
- Утечки данных (`current_salary`, `job_satisfaction`, `work_life_balance`) исключены из признаков модели автоматически через `_LEAKY_COLS`.
